Magic Ball Sorter

In teams of two, make your own magic eight ball with JavaScript. It should use an array to store the different magic eight ball outcomes, and Math.random to utilize the random choice. There should be a way for the user to start the magic eight ball.

One of you start a public repo for the project and the other person become a collaborator. When I look at track changes, I should be able to see that both people did work on it.

The assignment should obviously be on github then, and send me the link on Slack.
